Cgit
====

Sublime Text 2 plugin to browse code on cgit.

# Usage

Add setting to `Preferences.sublime-settings`:

```
{
  "cgit":
  {
    "base_url": "http://example.com/"
  }
}
```

or `.sublime-project`:

```
{
  "settings": {
    "cgit":
    {
      "base_url": "http://example.com/"
    }
  }
}
```

Execute `cgit_browse` command by using the Command Pallet.
